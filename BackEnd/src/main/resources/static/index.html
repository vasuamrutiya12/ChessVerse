Chess Game Backend is Running

WebSocket endpoint: ws://localhost:3000/

API endpoints:

  • POST /api/auth/google - Google authentication
  • GET /api/auth/check - Check authentication status
  • POST /game/sendfriendrequest - Send friend request
  • POST /game/handlefriendrequest - Handle friend request
  • POST /game/sendgamerequest - Send game request
  • POST /game/handlegamerequest - Handle game request
